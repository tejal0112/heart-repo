# Heart Disease Analysis Web Portal

This project is a comprehensive web application built using the Flask framework and Tableau visualizations to analyze critical heart disease risk factors such as BMI, Smoking, and Cholesterol levels.

## Key Features
- Interactive Dashboards: More than 8 sheets demonstrating demographic and lifestyle impacts on heart health.
- Professional UI: Dedicated sections for Home, About, and Analytics.
- Responsive Design: Fully optimized for various screen sizes and devices.

## Tech Stack
- Backend: Python (Flask)
- Frontend: HTML5, CSS3
- Visualization: Tableau Public
- Database: MySQL

## Project Structure
- app.py: Main Flask application file for routing and logic.
- templates/: Contains HTML files including index.html, about.html, and dashboard.html.
- static/: Directory for background images, icons, and CSS stylesheets.
